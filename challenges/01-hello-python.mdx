## โจทย์

เขียนโปรแกรมที่:

1. แสดงข้อความ `Hello, Python!` ออกมา
2. สร้างตัวแปร `name` เก็บชื่อของคุณ
3. แสดงข้อความ `สวัสดี, [ชื่อ]! ยินดีต้อนรับสู่ Python` โดยใช้ f-string

## ตัวอย่าง Output

```
Hello, Python!
สวัสดี, สมชาย! ยินดีต้อนรับสู่ Python
```

## Hints

- ใช้ `print()` สำหรับแสดงผล
- สร้างตัวแปร: `name = "ชื่อของคุณ"`
- ใช้ f-string: `f"สวัสดี, {name}!"`
