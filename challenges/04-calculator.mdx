## โจทย์

สร้างเครื่องคิดเลขที่:

1. มีฟังก์ชัน `add`, `subtract`, `multiply`, `divide`
2. ฟังก์ชัน `divide` ต้องจัดการกรณีหารด้วย 0 (return `"Error: หารด้วย 0 ไม่ได้"`)
3. ฟังก์ชัน `calculate(a, operator, b)` เรียกใช้ฟังก์ชันที่เหมาะสม

## ตัวอย่าง Output

```
10 + 5 = 15
10 - 3 = 7
6 * 7 = 42
20 / 4 = 5.0
10 / 0 = Error: หารด้วย 0 ไม่ได้
```

## Hints

- ใช้ `if operator == "+"` สำหรับเลือกการคำนวณ
- `try-except ZeroDivisionError` หรือตรวจสอบ `b == 0`
- ฟังก์ชันสามารถ return string หรือตัวเลขได้
