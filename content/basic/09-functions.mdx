## Return — ส่งค่าออก

`return` ส่งค่าออกจากฟังก์ชันในรูปแบบข้อมูล เพื่อนำไปใช้ต่อได้ (ต่างจาก print ที่แค่แสดงผล)

<CodeRunner code={`def multiply(a, b):
    return a * b

result = multiply(5, 3)
print("ผลลัพธ์:", result)
print("บวกเพิ่ม:", multiply(5, 3) + 20)`} />

<CodeRunner code={`def total(*args):

## Mutable Default Argument — กับดักสำหรับมือใหม่

<CodeRunner code={`# อันตราย! list เดิมถูกใช้ซ้ำทุกครั้ง
def bad_append(value, lst=[]):
    lst.append(value)
    return lst

print(bad_append(1))  # [1]
print(bad_append(2))  # [1, 2] !!!

# วิธีที่ถูกต้อง — ใช้ None แทน
def good_append(value, lst=None):
    if lst is None:
        lst = []
    lst.append(value)
    return lst

print(good_append(1))  # [1]
print(good_append(2))  # [2] ✓`} />

## Recursion — ฟังก์ชันเรียกตัวเอง

ฟังก์ชันที่เรียกใช้ตัวเองซ้ำๆ เพื่อแก้ปัญหาย่อย จนกว่าจะถึงเงื่อนไขหยุด (base case)

### Factorial

<CodeRunner code={`def factorial(n):
    if n == 0 or n == 1:  # base case
        return 1
    else:
        return n * factorial(n - 1)  # เรียกตัวเอง

print(f"5! = {factorial(5)}")
print(f"10! = {factorial(10)}")`} />

### Fibonacci

<CodeRunner code={`def fibonacci(n):
    if n <= 0:
        return "Invalid"
    elif n == 1 or n == 2:
        return 1
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)

for i in range(1, 11):
    print(f"F({i}) = {fibonacci(i)}")`} />

## Docstrings — เอกสารประกอบฟังก์ชัน

<CodeRunner code={`def bmi(weight: float, height: float) -> float:
    """คำนวณค่า BMI

    Args:
        weight: น้ำหนักเป็น kg
        height: ส่วนสูงเป็น m

    Returns:
        ค่า BMI (Body Mass Index)
    """
    return weight / (height ** 2)

result = bmi(70, 1.75)
print(f"BMI = {result:.1f}")

# ดู docstring
print(bmi.__doc__)`} />

## สรุป

- **def** สร้างฟังก์ชัน, **return** ส่งค่าออก (return หลายค่าได้)
- **Default parameters**: `def f(x, y=10):` — กำหนดค่าเริ่มต้น
- **Type hints**: `def f(x: int) -> str:` — บอกชนิดข้อมูล
- **`*args`** = tuple ของ positional args | **`**kwargs`** = dict ของ keyword args
- **ระวัง mutable default** — ใช้ `None` แทน `[]` หรือ `{}`
- **Recursion** — ฟังก์ชันเรียกตัวเอง ต้องมี base case เสมอ
- **Docstrings** — เขียนเอกสารประกอบด้วย `"""..."""`

บทต่อไป: **Lambda และ Higher-Order Functions** — ฟังก์ชันนิรนาม map filter sorted
