## ทำไมต้อง Loop?

สมมติว่าต้อง print คำว่า "tanakrit" 10 ครั้ง — ถ้าไม่มี loop ก็ต้องพิมพ์ print 10 บรรทัด

<CodeRunner code={`# แบบไม่ใช้ loop (ไม่ดี!)
print("tanakrit")
print("tanakrit")
print("tanakrit")
# ... ต้องพิมพ์อีก 7 รอบ

# แบบใช้ loop (ดีกว่า!)
for i in range(10):
    print("tanakrit")`} />

## For Loop

![For Loop Flowchart](/images/lessons/loops/for-loop-flowchart.png)

### `range()` แบบ 1 parameter — `range(stop)`

สร้างชุดตัวเลขตั้งแต่ 0 ถึง stop-1

<CodeRunner code={`for i in range(5):
    print(f"รอบที่ {i}")`} />

### `range()` แบบ 2 parameters — `range(start, stop)`

<CodeRunner code={`for i in range(5, 10):
    print(f"i = {i}")`} />

### `range()` แบบ 3 parameters — `range(start, stop, step)`

<CodeRunner code={`for i in range(0, 20, 3):
    print(f"i = {i}")`} />

### ตัวอย่าง — แม่สูตรคูณ

<CodeRunner code={`for i in range(12):
    print(f"5 x {i + 1} = {5 * (i + 1)}")`} />

## Loop in List

<CodeRunner code={`fruit_list = ["apple", "banana", "cherry"]

for fruit in fruit_list:
    print(f"ผลไม้: {fruit}")`} />

### `enumerate()` — ได้ทั้ง index และค่า

`enumerate()` ให้ทั้งลำดับและค่าในรอบเดียว ไม่ต้องใช้ `range(len(list))`

<CodeRunner code={`fruits = ["apple", "banana", "cherry", "grape"]

# แบบเดิม (ใช้ได้แต่ไม่สวย)
for i in range(len(fruits)):
    print(f"{i}: {fruits[i]}")

print("---")

# แบบ pythonic ด้วย enumerate
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# เริ่มนับจากเลขอื่นก็ได้
print("---")
for i, fruit in enumerate(fruits, start=1):
    print(f"{i}. {fruit}")`} />

### `zip()` — วนซ้ำหลาย list พร้อมกัน

<CodeRunner code={`names = ["สมชาย", "สมหญิง", "สมศักดิ์"]
scores = [85, 92, 78]

for name, score in zip(names, scores):
    print(f"{name}: {score} คะแนน")`} />

## Loop in Dict

<CodeRunner code={`car_dict = {
    "brand": "Ford",
    "model": "Mustang",
    "year": 1964,
}

# loop เอา key
for key in car_dict:
    print(key)

print("---")

# loop เอา key + value ด้วย .items()
for key, value in car_dict.items():
    print(f"{key}: {value}")`} />

## Nested Loop — ลูปซ้อนลูป

### สร้างรูปดาวสามเหลี่ยม

<CodeRunner code={`for i in range(5):
    for j in range(i + 1):
        print("*", end="")
    print("")`} />

### สร้างรูปดาวกลับหัว

<CodeRunner code={`for i in range(5):
    for j in range(5 - i):
        print("*", end="")
    print("")`} />

### แม่สูตรคูณ 1-12

<CodeRunner code={`for i in range(1, 4):  # แสดงแค่แม่ 1-3 เป็นตัวอย่าง
    print(f"--- สูตรคูณแม่ {i} ---")
    for j in range(1, 13):
        print(f"{i} x {j} = {i * j}")
    print()`} />

## While Loop

![While Loop Flowchart](/images/lessons/loops/while-loop-flowchart.png)

`while` วนซ้ำไปเรื่อยๆ ตราบใดที่เงื่อนไขเป็น True

<CodeRunner code={`count = 0
while count < 5:
    print(f"count = {count}")
    count += 1
print("จบ loop")`} />

### `break` — หยุด loop ทันที

<CodeRunner code={`# จำลองเกมทายเลข
secret = 7
guesses = [3, 5, 7, 9]

for guess in guesses:
    print(f"ทาย: {guess}")
    if guess == secret:
        print("ทายถูกต้อง!")
        break
    else:
        print("ผิด ลองใหม่")`} />

### `continue` — ข้ามรอบนี้ไปทำรอบถัดไป

<CodeRunner code={`# แสดงเฉพาะเลขคี่
for i in range(1, 11):
    if i % 2 == 0:
        continue  # ข้ามเลขคู่
    print(i)`} />

    if i % 2 == 0:

## List Comprehension (แนะนำสั้นๆ)

วิธีสร้าง list จาก loop ในบรรทัดเดียว (จะเรียนเต็มๆ ในบทที่ 8)

<CodeRunner code={`# แบบ loop ปกติ
squares = []
for x in range(10):
    squares.append(x ** 2)
print(squares)

# แบบ list comprehension (บรรทัดเดียว!)
squares2 = [x ** 2 for x in range(10)]
print(squares2)`} />

## สรุป

- **for loop** — วนซ้ำตามจำนวนที่กำหนด ใช้กับ `range()`, list, dict
- **while loop** — วนซ้ำตราบใดที่เงื่อนไขเป็น True
- **break** = หยุด loop | **continue** = ข้ามรอบนี้
- **enumerate()** — ได้ทั้ง index และค่า
- **zip()** — วนซ้ำหลาย list พร้อมกัน
- **Nested loop** — ลูปซ้อนลูป เหมาะกับข้อมูล 2 มิติ

บทต่อไป: **โครงสร้างข้อมูล** — List, Tuple, Dictionary, Set
